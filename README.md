# AI-OS Scripts

Aktualisierbare Skripte für AI-OS (Kubuntu 24.04 Basis).

## Update auf installiertem System

```bash
sudo ai-os-update
```

## Enthaltene Skripte

| Script | Funktion |
|--------|---------|
| `ai-setup` | Ersteinrichtung: AGB → KI-System → Memory → Tresor |
| `ai-agb` | Nutzungsbedingungen anzeigen |
| `ai-chat` | KI-Chat mit Memory und Datenschutz-Hinweis |
| `ai-memory-update` | Memory nach Chat automatisch aktualisieren |
| `ai-os-update` | Dieses Update-Script |
| `tresor-create` | LUKS-Tresor erstellen |
| `tresor-open` | Tresor öffnen |
| `tresor-close` | Tresor sperren |
| `tresor-status` | Status + Audit-Log anzeigen |
